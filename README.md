# daodaoExcel
    基于canvas的和js开发的类excel表格，支持用户自定义添加工具栏以及右键菜单

## 基本使用

### 初始化
    引入daodaoExcel dist目录中的js文件
    示例：
    >    <div id="daodao" style="width:1000px;height:600px;"></div>
    >    <script src="daodaoExcel.js"></script>
    >    <script>
    >        var excel = new DaoDaoExcel({id:'daodao'})
    >    </script>
